# 易宿酒店预订平台 · PC端管理系统

一个完整的酒店管理后台，包含商户端和管理员端。

---

## 技术栈

- **前端**：React 18（CDN加载，无需构建工具）
- **后端**：Node.js + Express
- **数据库**：SQLite（better-sqlite3）

---

## 快速启动

### 第一步：安装后端依赖

```bash
cd backend
npm install
```

### 第二步：启动后端服务

```bash
node server.js
# 服务运行在 http://localhost:3001
```

### 第三步：打开前端

直接用浏览器打开 `frontend/index.html` 文件即可。

> 注意：前端需要后端服务运行才能正常使用。

## 功能说明

### 商户端
- ✅ **注册/登录**：支持商户角色注册
- ✅ **酒店信息录入**：填写酒店名(中/英)、地址、星级、开业时间、设施、周边景点
- ✅ **房型管理**：支持多房型、原价/优惠价、优惠说明（折扣/套餐）
- ✅ **草稿保存**：可先保存草稿，再提交审核
- ✅ **编辑修改**：草稿和已拒绝的酒店可以编辑
- ✅ **提交审核**：一键提交，等待管理员审核

### 管理员端
- ✅ **注册/登录**：支持管理员角色注册
- ✅ **审核管理**：查看所有商户提交的酒店
- ✅ **状态筛选**：按待审核/已发布/已拒绝/已下线筛选
- ✅ **审核通过**：将酒店状态变为已发布
- ✅ **拒绝并说明原因**：填写拒绝理由，商户可看到
- ✅ **下线/恢复上线**：支持下线不删除，可恢复

---

## 酒店状态流转

```
草稿(draft) → 待审核(pending) → 已发布(approved) ↔ 已下线(offline)
                              ↘ 已拒绝(rejected) → 草稿(编辑后重新提交)
```

---

## 数据库结构

- `users`：用户表（商户/管理员）
- `hotels`：酒店信息表
- `room_types`：房型价格表（关联酒店）
